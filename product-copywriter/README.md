# ✍️ AI Копирайтер

Генерация продающих текстов для Instagram, Kaspi и сайта по фото товара — за 10 секунд.

## Стек

| Библиотека | Зачем |
|---|---|
| Python 3.10+ | Среда выполнения |
| Streamlit | Веб-интерфейс без фронтенда |
| OpenAI GPT-4o Vision | Анализ изображения + генерация текста |
| Pillow | Ресайз изображения перед отправкой в API |

## Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd product-copywriter

# 2. Создать виртуальное окружение
python -m venv .venv

# Windows
.venv\Scripts\activate
# Mac/Linux
source .venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Добавить API-ключ
mkdir .streamlit
echo 'OPENAI_API_KEY = "sk-..."' > .streamlit/secrets.toml

# 5. Запустить
streamlit run app.py --server.headless true
```

Приложение откроется на http://localhost:8501

## Использование

1. Загрузи фото товара (JPG / PNG / WEBP)
2. Посмотри превью — убедись что фото загрузилось корректно
3. Введи ключевые слова или краткое описание товара
4. Нажми **⚡ Сгенерировать тексты**
5. Переключайся между вкладками **Instagram / Kaspi / Сайт**
6. Скачай результат в JSON одной кнопкой

## Функции

- **Sidebar с FinOps** — показывает токены и стоимость каждого запроса в реальном времени (цены GPT-4o-mini: $0.15/1M input, $0.60/1M output). Накапливает суммарные расходы за сессию с кнопкой сброса.
- **Табы результатов** — Instagram, Kaspi и Сайт в отдельных вкладках, не нужно скроллить.
- **Превью изображения** — отображается сразу после загрузки, до генерации.
- **JSON Mode** — модель возвращает валидный JSON или приложение показывает понятную ошибку.
- **Скачать JSON** — один клик, все три варианта текста в одном файле.

## Безопасность

- API-ключ хранится в `.streamlit/secrets.toml` — читается через `st.secrets`, не через переменные окружения
- Файл добавлен в `.gitignore` — ключ никогда не попадает в репозиторий
